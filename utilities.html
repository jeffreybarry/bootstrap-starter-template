Utilities

Quick styles using Bootstrap

Use this page as a guide for learning more about Bootstrap utility classes that you might commonly use. The links below take you to the Bootstrap documentation for that utility. Only a few Bootstrap utilities are listed below. See the Bootstrap documentation for other utilities.

Bootstrap utilities are pre-defined CSS classes that you can add to your HTML. These Bootstrap CSS classes allow you to quickly make small, stylistic adjustments to your web page. While you can do all these things by writing your own CSS code, these Bootstrap utilities make a few of these tasks much quite easy: just add the Bootstrap utility class name to the HTML element you want to style.

Borders

Border color and border radius are very useful for modifying the borders around images or boxes. Hint: images often look better when a thin black border.

Adding a border radius allows you to change the corners of a border to rounded corners. You use border radius to turn an element into a circle.

Shadows

The shadow class utility allows for adding a shaded area, a shadow, around parts of an element. See the Bootstrap shadows utility documentation for examples.
